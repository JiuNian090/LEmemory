# LEmemory 微信小程序背记工具 - 项目创建计划

## 一、项目概述
根据 project.md 文档，创建一个基于微信小程序原生开发（TypeScript + 微信云开发）的背记工具，包含卡牌组管理、学习、统计等功能。

## 二、当前项目状态分析
当前已有微信小程序 TS-基础模版结构，包含：
- miniprogram/ 主目录
- pages/index 和 pages/logs 示例页面
- app.ts、app.json、app.wxss 入口文件
- typings/ 类型定义
- 基础项目配置文件

## 三、实施步骤

### 1. 目录结构调整
- 删除 pages/index 和 pages/logs 示例页面
- 创建项目文档要求的 pages、components、utils、images、cloudfunctions 目录

### 2. 基础配置文件更新
- 更新 app.json：添加页面路径、配置 tabBar（学习/统计/我的）
- 更新 app.ts：初始化小程序、配置云开发
- 更新 app.wxss：全局样式
- 添加 sitemap.json

### 3. 类型定义
- 创建 utils/types.ts：定义用户、卡牌组、卡牌、学习记录等全局 TypeScript 类型

### 4. 工具函数
- 创建 utils/db.ts：云数据库操作封装
- 创建 utils/time.ts：时间格式化、计时工具

### 5. 页面开发
按以下顺序创建页面：
- pages/mine（我的页面）：用户信息、登录
- pages/study（学习页面）：卡牌组列表、创建
- pages/cardDetail（卡牌组详情）：学习、目录、卡牌管理、收藏
- pages/statistics（统计页面）：学习数据图表

### 6. 组件开发
- components/flipCard：翻牌组件
- components/cardGroupItem：卡牌组列表项组件
- components/ec-canvas：ECharts 图表组件

### 7. 静态资源
- 创建 images/ 目录，准备图标等静态资源

### 8. 云函数目录
- 创建 cloudfunctions/ 目录及子目录（后续部署云函数使用）
